    16. HTML Favicon

    Favicons are small icons that are used to represent a website or web page in the browser's address bar, bookmarks, and tabs. They are an important part of a website's branding and can help to make it more recognizable and memorable to users. In this blog post, we will explore how to add a favicon to an HTML document.

    What is a Favicon?

    A favicon is a small icon that is associated with a website or web page. It is typically displayed in the browser's address bar, bookmarks, and tabs. Favicons can be in the form of an image file, such as a PNG or ICO file, and are usually square in shape with dimensions ranging from 16x16 to 64x64 pixels.

    How to Add a Favicon to an HTML Document

    Adding a favicon to an HTML document is a simple process that involves adding a link tag to the head section of the HTML document. Here is an example of how to add a favicon to an HTML document: 

    

    <head>
  <link rel="icon" href="favicon.ico" type="image/x-icon">
</head>

    In this example, we use the <link> tag with the rel attribute set to "icon" to specify that we are linking to a favicon. The href attribute specifies the location of the favicon file, and the type attribute specifies the MIME type of the favicon file.

    The favicon file can be in any format that is supported by the browser, such as PNG, GIF, or ICO. However, ICO is the most widely used format for favicons.

    It is important to note that the favicon file should be placed in the root directory of the website or web page. This is because the browser looks for the favicon file in the root directory by default.

    In addition to the <link> tag, there are other ways to add a favicon to an HTML document. For example, the <meta> tag can be used to specify the favicon file in the head section of the HTML document: 

    

    <head>
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="shortcut icon" href="/favicon.ico" type="image/x-icon">
</head>

    In this example, we use the <meta> tag with the name attribute set to "viewport" to specify the viewport settings for the web page. We also use the <link> tag with the rel attribute set to "shortcut icon" to specify the favicon file.
