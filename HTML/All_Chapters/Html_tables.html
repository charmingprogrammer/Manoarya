    10. HTML Tables

    HTML tables are a powerful tool for displaying and organizing data on a web page. They allow you to present information in a clear and structured way, making it easy for users to understand and navigate. In this blog post, we will cover some of the basics of working with HTML tables.

    1. Creating a Table

    To create a table in HTML, you need to use the <table> tag. Within this tag, you can create rows using the <tr> tag and cells using the <td> tag. You can also add a heading row using the <th> tag.

    Example:

    <table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Location</th>
  </tr>
  <tr>
    <td>John Doe</td>
    <td>30</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Jane Smith</td>
    <td>25</td>
    <td>London</td>
  </tr>
</table>
    1. Merging Cells

    You can merge cells in an HTML table using the colspan and rowspan attributes. The colspan attribute allows you to merge cells horizontally, while the rowspan attribute allows you to merge cells vertically.

    Example:

    <table>
  <tr>
    <th>Name</th>
    <th colspan="2">Age and Location</th>
  </tr>
  <tr>
    <td>John Doe</td>
    <td rowspan="2">30</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Jane Smith</td>
    <td>London</td>
  </tr>
</table>
    1. Adding Borders

    You can add borders to an HTML table using the border attribute. This attribute specifies the width of the border in pixels.

    Example:

    <table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Location</th>
  </tr>
  <tr>
    <td>John Doe</td>
    <td>30</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Jane Smith</td>
    <td>25</td>
    <td>London</td>
  </tr>
</table>
    1. Styling Tables

    You can style HTML tables using CSS. This allows you to customize the appearance of the table, including the font, color, and background.

    Example:

    table {
  border-collapse: collapse;
  width: 100%;
  font-family: Arial, sans-serif;
  color: #333;
}

th, td {
  padding: 8px;
  text-align: left;
  border-bottom: 1px solid #ddd;
}

th {
  background-color: #f2f2f2;
}

    In conclusion, HTML tables are a powerful tool for displaying and organizing data on a web page. By using the <table>, <tr>, <td>, and <th> tags, you can create a structured and easy-to-read table. You can also merge cells, add borders, and style the table using CSS. By using HTML tables effectively, you can create a user-friendly and engaging web experience for your audience.
