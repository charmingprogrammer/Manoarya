    13. HTML Id

    HTML ID is a crucial concept in web development that allows you to identify and manipulate specific elements on a web page. In this blog post, we will cover the basics of working with HTML IDs.

    What is an HTML ID?

    An HTML ID is a unique identifier that you can assign to an element on a web page. This ID is used to identify the element in JavaScript, CSS, and other web development tools. Unlike HTML classes, which can be applied to multiple elements, IDs must be unique within a web page.

    How to Define an HTML ID?

    To define an HTML ID, you need to use the id attribute. The value of the id attribute is the name of the ID that you want to assign to the element.

    Example:

    

    <div id="header">
  <h1>Welcome to my website!</h1>
</div>

    In this example, we have defined an ID called "header" and applied it to a <div> element.

    How to Access an HTML Element by ID in JavaScript?

    To access an HTML element by ID in JavaScript, you can use the getElementById() method. This method takes the ID of the element as a parameter and returns the element object.

    Example:

    

    let header = document.getElementById("header");
header.innerHTML = "Welcome to my new website!";

    In this example, we have accessed the element with the ID "header" using the getElementById() method and updated its inner HTML to display a new message.

    How to Style an HTML Element by ID in CSS?

    To style an HTML element by ID in CSS, you can use the ID selector (#). This selector selects the element with the specified ID and applies the specified styles.

    Example:

    

    #header {
  background-color: #333;
  color: #fff;
  font-size: 24px;
}

    In this example, we have selected the element with the ID "header" using the ID selector (#) and applied a background color of #333, a text color of #fff, and a font size of 24px.

    In conclusion, HTML IDs are a powerful tool in web development that allow you to identify and manipulate specific elements on a web page. By using the id attribute and JavaScript and CSS, you can create dynamic and engaging web experiences for your audience. By understanding HTML IDs, you can take your web development skills to the next level and create web pages that stand out from the crowd.
