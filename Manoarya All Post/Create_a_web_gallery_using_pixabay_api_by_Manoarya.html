      Now time for Javascript Code which creates all functionality enabled.

      First of all we access all necessary elements from Dom and store it in a variable. 

      							
// fetch small image container
const allImg = document.getElementById('allImg');

      Now we create an async and await function for fetching API data.

      							
async function getData() { 
const responseData = await fetch('https://pixabay.com/api/?key=21480493-5c466af89bb6c5ee2417d3e5b&q='+topic+'&image_type=photo&pretty=true'); 

const MainData = await responseData.json(); 

const hits = MainData.hits;
allImg.innerHTML = ``;
document.getElementById('img_changer').innerHTML = `<img src="${hits[1].largeImageURL}" alt="main post" />`;

//loop all MainData 
for (let i = 0; i < hits.length; i++) {
allImg.insertAdjacentHTML('afterbegin',`<li><a href="${hits[i].largeImageURL}"><span id="tags">${hits[i].Tags}</span><img src="${hits[i].previewURL}" alt="${hits[i].Tags}" /></a></li>`); 
} 
}

      For API you need to go to pixabay.com and make a API key and then you can use it.

      Learn About API

      Now you copy your own API key from pixabay and pasted it inside of await fetch('Past API Hare');

      In the topic section you can choose any topic like nature, football...etc.

      Now we create a const and store all data which we fetch early and convert this data into json format.

      Now we need an array to show all data on the Dom which we created earlier.

      For that create a new const and store array like this Const myArray = MainData.hits;

      Now let's loop all array data using a for loop and insert necessary information using InsertAdjacentHTML(Not remove any data from Dom and we can use a lot of content using this.) like shown inside of the blue box in the image below .

      After that we called the async function.

      Now call an onkeyup event which we created inside of the search input tag into the html section.

      Inside of this function we create a const and store our search input.

      And now match the topic value with the search input value.

      So that API knows what topic the user is searching and shows results according to that topic.

      And very important we call an async function every time when a user searches for anything.

      And update the async function to show content according to user search.

      You can see this image for better understanding.

      Now for better user experience we add a loader until data fetching.

      For that we add a loader gif before async function. And after that we clear that loader when the async function is called.
