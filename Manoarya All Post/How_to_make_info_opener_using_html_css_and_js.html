      Environment Setup

      In this section we are going to set up our coding environment which helps us to create an info opener.

      1. Install VS code if you haven't

      Visual studio code is the cross platform powerful code editor which is used by billions of developers every day.

      We used visual studio code to write our code fast and error free.

      Install Visual Studio Code

      2. Create a folder

      Create a folder on any directory I suggest you create on the desktop directory and inside of this folder create three files named index.html, style.css and script.js.

      You can give the name whatever you want but the extension of the file is respectively .html, .css and .js.

      3. Open Index.html file

      Open the index.html file on the current window and start implementing the code step by step.

      (i) First add html Starter Code

      For Adding starter code just press shift + ! This shortcut helps us to add html starter code in visual studio code.

      (ii) Add H1

      Now add h1 heading inside of the body which exists on the top of the page.

      (iii) Add buttons After the h1 heading now add buttons which helps us to open information which exists inside of paragraph. Each button tag contains an onclick event.

      Inside of each button add button name and after that add a span tag which helps us to indicate state of open and close. Each span tag contains a unique id.

      (iv) Add paragraph

      After each button add a paragraph which contains our information. And open it using a button.

      4. Open script.js file

      Open script.js file into a current window and start implementing code step by step. (i) Get all the element from html

      Get all the necessary elements from html and store it into a variable.

      //get info
      const info1 = document.getElementById('info1');
      const info2 = document.getElementById('info2');
      const info3 = document.getElementById('info3');
      //get span
      const icons1 = document.getElementById('icons1');
      const icons2 = document.getElementById('icons2');
      const icons3 = document.getElementById('icons3');

      (ii) Make a functions

      Now make a function so when we click on the button then onclick event trigger and add the style that we want to add using the classList.toggle.

      classList.toggle helps us to add a new class on the existing html elements.

      When the new class is added then we style that class and apply style on the element.

      Final Javascript Code

             
//get info
    const info1 = document.getElementById('info1');
    const info2 = document.getElementById('info2');
    const info3 = document.getElementById('info3');
//get span + -
    const icons1 = document.getElementById('icons1');
    const icons2 = document.getElementById('icons2');
    const icons3 = document.getElementById('icons3');
//onclick function
    function info1btn() {
      info1.classList.toggle('open');
      icons1.classList.toggle('icons');
    }
    function info2btn() {
      info2.classList.toggle('open');
      icons2.classList.toggle('icons');
    }
    function info3btn() {
      info3.classList.toggle('open');
      icons3.classList.toggle('icons');
    }
