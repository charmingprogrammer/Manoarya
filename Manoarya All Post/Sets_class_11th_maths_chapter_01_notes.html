What is sets ?

Set is a well defined collection of objects.

Other meaning of sets : Collection, Aggregate, class etc.

Sets denotes by capital letters like A, B, C, D etc.

Elements denotes by small letters like a, b, c, d etc.

Example : The collection of vowel in english alphabet.
Answer : This sets contains five elements a, e, i, o, u.

Reserve letters for sets

N : For the set of natural numbers.
Z : For the set of integers.
Z+ : For the set of positive integers.
Q : For the set of all rational numbers.
Q+ : For the set of all positive rational numbers.
R : For the set of all real numbers.
R+ : For the set of all positive real numbers.
C : For the set of all complex numbers.

Descriptions of sets

Set described in 2 terms.

1. Roster Form or tabular from
2. Set-builder form

1. Roster Form or tabular Form

In this form a set is describe by listing elements, separated with commas(,) within bracket{}.

Example : The set of vowels in english alphabet.
In Roster from : {a, e, i, o, u}

Set-Builder form

In this form a set is described by characterizing property p(x) of its elements x.

Simple structure of Set-builder form

{x:P(x)holds}
{x|P(x)holds}

'|' or ':' symbol read as 'Such that'

Example : The set E of all even natural numbers.
In Set-builder form : E = {x:x is a natural numbers and x = 2n for n∈N}

Types of sets

1. Empty Set 2. Singleton Set 3. Finite Set 4. Infinite Set 5. Equivalent set 6. Equal Set

1. Empty Set :

A set is said to be empty or null or void set if it has no elements.

Empty Sets denoted by (symbole).

Empty sets denoted in roster form like this {}. Example : {x}

2. Singleton Sets :

A set consisting of a single element is called a singleton set.

Example : The set {5} is a singleton set.

3. Finite Sets :

A set called finite set if it is either void set or ites element can be listed (counted) by natural numbers 1,2,3...and the process of listing terminates at a certain natural number (n).

Each one of the following is a finite set :

1. Set of even natural numbers less than 100.
2.Set of soldiers in the Indian army.
3. Set of all our people on the earth.

What is a cardinal number ?

A number denoting quantity (one, two, three, etc.), as opposed to an ordinal number (first, second, third, etc.).

Cardinal numbers of finite sets : The number ‘n’ in the above definition is called the cardinal number or order of a finite ‘A’ and its denoted by n(A).

4. Infinite Set :

A set whose element cannot be listed by natural numbers 1,2,3….n, for any natural number ‘n’ is called infinite sets.

Each one of the following sets is infinite set :

1. Set of all points in a plane.
2. Set of all lines in a plane.
3. {x ∈ R : 0 < x < 1}

5. Equivalent Sets :

Two finite sets ‘A’ and ‘B’ and equivalent if their cardinal numbers are the same.

Example : n(A) = n(B)

6. Equal Set :

Two sets ‘A’ and ‘B’ are said to be equal if every element of ‘A’ is a member of ‘B’, and every element of ‘B’ is a member of ‘A’.

When ‘A’ and ‘B’ are equal then we write like this A = B .
And if not equal then we write like this A =/= B.

Example :
A = {5, 2, 1, 6}
B = {6, 5, 2, 1}

Set ‘A’ and ‘B’ are equal sets right !
So, A = B

Different Between equivalent set and equal set.

A = {5,2,1,6}
B = {6,5,2,1}

Set ‘A’ and ‘B’ have the same number of elements and same elements so both sets are equivalent and equal sets.

Look at the following example :

C = {1,2,3,4}
D = {a,b,c,d}

Sets ‘C’ and ‘D’ have the same number of elements, that's why sets ‘C’ and ‘D’ are equivalent sets.

But Sets ‘C’ and ‘D’ are not equal sets because both do not have the same elements.

Equal sets are equivalent set but equivalent sets are not equal sets.

Subset :

Let A and B be two sets if every element of A is an element of B then A is called a subset of B.

If A is a subset of B then we write like this : A ⊆ B and read as “ A is a subset of B ” or “A is contained in B”.

Super set : If A is a subset if B, we say that B contains A or B is a superset of A and we write B⊃A.

If A is not a subset of B, we write A⊄B.

Every set is a subset of itself.

Empty sets are subsets of every set.

For Example : {1} = {1,2,3,4,5}

Universal Set :

A set that contains all sets in a given context is called that universal set.

Example : When we are using sets containing natural numbers, then N is the Universal Set.

A = {1,2,3}
B = {2,4,5,6}
C = {1,3,5,7}

Universal Set (U) = {1,2,3,4,5,6,7}

Universal set denoted by ‘U’.

Power Set :

Let A be a set. then the collection of family of all subsets of A is called the power set of A and is denoted by p(A).

that is , P(A) = { S : S ⊂ A }

Power set of a given set is always non-empty.

Venn Diagrams :

In venn diagrams the universal set(U) is represented by points within a rectangle and its subsets are represented by points in closed curves (usually circle) within the rectangle.
