Hi ! What's up, how is it going?
In this post we are going to develop this beautiful website using just simple html, css and js.

In this project you learn about following terms.

1. Know about responsive meta tags.
2. Learn responsive layout making.
3. Learn how to link stylesheets or css files.
4. Learn how to use internal Javascript.
5. Learn how to use beautiful icons.
6. Learn about list items.
7. Learn about Flex.
8. Learn how media query works.
9. Learn how to use an image as a background.
10. Learn about js functions.
11. Learn how onscroll events work.

1. HTML starter code

          
            <!DOCTYPE html>
<head>
  <meta http-equiv="content-type" content="text/html; charset=utf-8" />
  <!--responsive meta -->
  <meta name="viewport"
  content="width=device-width, initial-scale=1.0">
  <!-- Theme Title-->
  <title> Add Title</title>
  <!--meta description-->
  <meta name="description" content="Add description" />
</head>
<body>
  
</body>

Now you Add your own title inside of title tag which is represent your whole website. For this project I am gonna add Theme Design 01 by Manoary as a title. you can add whatever you want.

Now instead of Add Descriptions you need to put your own Descriptions means what your website do or what services your website provides. In this project I am gonna use This Design is made by Manoarya.as a Descriptions.

2. Link CSS file.

            <!--link css file-->
  <link rel="stylesheet" href="Add CSS file" type="text/css" media="all" />

Now copy the above code and put inside of head tag and then change the Add CSS file with your css file name. I use Theme_Design_01_by_manoarya.css as a css file name.

3. Link fontawesome 

          
            <!--fontawesome for beautiful fonts-->
  <script src="https://kit.fontawesome.com/9e0ab445c3.js" crossorigin="anonymous"></script>

Copy the above code and put inside of the head tag. now you able to use fontawesome fonts.

5. Javascript Code 

          
            <!--Internal js -->
  <script type="text/javascript" charset="utf-8">

    const sidebar_container = document.getElementById('sidebar_container');

    function SidebarOpener() {
      sidebar_container.style.left = "0%";
    }
    function SidebarCloser() {
      sidebar_container.style.left = "-100%";
    }

    //change property using onscroll
   
            var prevScrollpos = window.pageYOffset; 
    window.onscroll = function() {
      var currentScrollPos = window.pageYOffset; 
      if (prevScrollpos > currentScrollPos) {
        document.getElementById("nav").style.background = "";
      } else {
        document.getElementById("nav").style.background = "#5e94e0";
      } prevScrollpos = currentScrollPos;
    }
  </script>
