Deploy the react web app on the github using 4 steps by manoarya.

    Category : HTM

    Hi! folks in this post we goana talk about how to deploy a react web apps on the github using Four simple steps. I hope you are exited to read this post. so let's start.

    I hope you have a prebuild react web app. if not then you can build it or get help form hare.

    1. Create a Repository on github.

    So in this step we gonna create a repository on github. so let's open github and sign in. And create a new repository.

    Open Github

    Ones you are signed in, click on the new repository button. now other page open fill the repository name and scroll down and hit create repository button.

    After clicking the repository button, you will see a new page with lots of information. I suggest you to copy the first link of the page.

    2. Run Some Comamnds

    Now back to the vs code or whatever IDE you are using. open the terminal in your react app directory and run the following commands.

    1. git init
    2. git remote add origin {Paste URL link that you copy before without curly bracks.}
    3. yarn add -D gh-pages

    3. Edit package.json file.

    Now go the the package.json file and add following

    {
    "homepage": "http://{Your github user name}.github.io/{Repository Name}",
    "scripts": {
    "deploy": "gh-pages -d build",
    "predeploy": "npm run build"
    }
    }

    4. Yarn deploy

    Now finally run yarn deploy command.

    I hope you learn how to deploy the react app on the github pages with live url. now you can access your react website on the url that you put on the package.json homepage object.
