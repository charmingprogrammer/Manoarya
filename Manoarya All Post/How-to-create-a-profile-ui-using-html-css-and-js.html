Let's Started.

See Demo

Fist of all you need to create a file and name it whatever you want.

Now Copy the following codes

Add HTML Code

          
            <!DOCTYPE html>
<head>
  <meta http-equiv="content-type" content="text/html; charset=utf-8" />
  <!--Add title that show on the browser header-->
  <title>User Profile</title>
</head>
<body>
  <div class="main_container">
    <div class="profile_container">
      <img class="background_img" src="https://cdn.pixabay.com/photo/2018/08/21/23/29/forest-3622519_1280.jpg" alt="background" />
      <img class="user_img" src="https://simgbb.com/avatar/M5zLLq7PnVGF.png" alt="user img" />
      <h1>Manohar Anand</h1>
      <p>
        Hi,This is Manohar Anand Founder of Manoarya.com + Founder of HCC + Teacher and Entrepreneur.
      </p>
    </div>
  </div>
</body>

Add CSS Code

          
    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
      font-family: Roman;
    }
    .main_container {
      height: 100%;
      width: 100%;
    }
    .profile_container {
      height: 500px;
      width: 100%;
      background: white;
      display: flex;
      flex-direction: column;
      justify-content: flex-start;
      align-items: center;
    }
    .background_img {
      height: 250px;
      width: 100%;
      transform: scale(0.9);
      border-top-left-radius: 10px;
      border-top-right-radius: 10px;
    }
    .user_img {
      height: 200px;
      width: 200px;
      border-radius: 100px;
      border: 6px solid white;
      margin-top: -100px;
      z-index: 123;
    }
    .profile_container h1 {
      background: none;
      color: #333333;
      font-size: 25px;
    }
    .profile_container p {
      background: none;
      text-align: center;
      transform: scale(0.9);
      font-size: 20px;
      font-weight: 400;
      color: #383838;
    }
