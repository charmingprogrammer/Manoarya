      HTML images

      The <img /> tag helps us to display the image on the web. These are some of the attributes used in the <img /> tag.

      Attributes of <img /> tag

      src : src attribute helps us to give the source of image. 

                  
<img src="https://cdn.pixabay.com/photo/2019/01/31/20/52/web-3967926_960_720.jpg">

      alt : alt tag is shown when the source of the image is not available.

      alt is very important for SEO(Search Engine Optimization).
      alt also helps the screen reader to read images.
      It is the alternative text of this image. 

                  
<img src="" alt="HTML image">

      Height : Height attribute used to define the height of the image.

      width : Width attribute used to define the width of the image.

      Border : Border attribute used to give the border from all sides. By default the border length is 0 in the img tag.
      And by default the color of the border is black.

      The final code of the img tag and its output. 

                  
<img src="https://cdn.pixabay.com/photo/2019/01/31/20/52/web-3967926_960_720.jpg"  alt="HTML image" height="100%" width="700px" border="10px">

      Output

      Formatted text in HTML

      <i> : used to display italic text in web browsers.

      <b> : Displays bold text on web browsers. <strong> : Displays bold text to emphasize it.

      <em> : display an italic text and emphasize it.

      <Sub> : Define the text on the bottom of the main text. Like Co2.

      <Sup> : Define the text on the top of the main text. Like 1002.

      <small> : Display the small size text on the web browser.

      <del> : Display the text which looks like deleted text. Means a line appears through the text.

      <ins> : Display the text which looks like inserted text. Usually these types of text are underlined.

      <blockquote>: This tag specifies that one section of text must be enclosed within a blockquote.

      <q> : This tag is used when you want to add quotes inside of a line.

      <cite> : used to cite the author of a quote.

      <address> shows the author's contact information.

      <abbr> : denotes an abbreviation meaning the shortcut of something.

      <code> : This tag is used when you want to show some code on the web browser.

      <mark> : Display highlighted text.

      HTML Links

      <a> tag is also known as anchor tag which helps us to define a hyperlink to jump to the other pages or the specific section of the same page.

      <a> tag text is underlined and the color of text is blue.

      Following are some attributes of the <a> tag.

      href : this attribute contains a url of other pages or the id of some section. When a user clicks on this then users jump to the targeted URL(Uniform Resource Locator) or the targeted section.

      download : this attribute helps us to download the targeted url content.

      target : this attribute controls the opening of any url. It would be opened on the same window or other windows. 

        
    <a href="https://manoarya.com">Open Manoarya </a>

      Forms in HTML

      Form in HTML <form> tag helps us to get input from the user and it creates a form inside of a html document.

      Following are attributes of <form> tag.

      action : specify where the form input data goes when the submit button is clicked.

      target : specifies where the form's response will appear.

      autocomplete : is either on or off.

      novalidate : Show that the form has not been validated.

      method : Indicates the HTTP method to use when sending form data.

      name : specify the name of the form.

      required : confirm that the input used into the form will not empty.

      autofocus : When the page is loaded, the input tag is focused.

      disable : disable the input tag. So that the user can interact with the user.

      placeholder : give the hint to the user so that the user understands what types of information will fill inside of the input tag.
