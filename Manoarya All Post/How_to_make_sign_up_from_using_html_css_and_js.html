    How to make Sign Up and Login from using HTML, CSS and Js? for free.

    Category : HTM

    Hi folks in this post we are going to build an awesome looking sign-up and login form using html, css and js.
    The best part of this build is you can easily toggle between sign from and login from.

    A step by step guide to create a signup and login form using html, css and javascript.
    You can jump to the specific section using the following list.

    See Demo


      Introduction

      Sign-up allows us to collect user information and store it into a database for better marketing and best services. And login allows us to inter on these services and use these services. Login is also known as sign-in.

      Several websites are integrated with sign-up and login forms on the Internet.

      So, I hope you are excited to build this amazing sign-up and login form step by step.

      So first set-up the development environment .

      Set-Up Environment.

      Let's open our visual studio code and create three files first index.html, second style.css and third script.js.

      Opening index.html file

      Open index.html file on the current window and start implementing code step by step.

      Insert html boilerplate

      Press shift + ! for inserting html starter code our starter boilerplate looks like below code. 

                
          <!DOCTYPE html>
      <html lang="en">
      <head>
         <meta charset="UTF-8">
         <meta http-equiv="X-UA-Compatible" content="IE=edge">
         <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Document</title>
      </head>
      <body>
        
      </body>
      </html>

      Change the title of this boilerplate. Instead of Document add Signup Form or whatever you want to add as a title. Title is the very important part of any html document. It appears at the top of any browser.

      Add Main Code

      Add a div with the id container inside the body tag. This div acts as a container. Inside of this container add one more div which acts as a container of two buttons. Now inside of this container add two buttons which helps to toggle between sign-up and login.

      Our body code looks like the code below. 

                  
              <body>
         <div class="container">
             <div id="fromBtns">
                 <button id="signUpBtn">Sign Up</button>
                 <button id="loginBtns">Login</button>
             </div>
             <!--Insert login From using js-->
             <div id="FormContainer">
             </div>
         </div>
      </body>

          font-family: Roman;

              width: 100%;

      Adding style on the input tags

            #FormContainer button:active {
              transform: scale(0.97);

      Open script.js file

      Now open script.js file into the current window and start implementing scripts step by step.

      Get All Elements from Dom

      Letâ€™s access all the elements from the index.html file and store it into a variable.
      We access three elements from dom.

      1. Sign up Button
      2. Login Button
      3. Form Container 

                  
              //access necessary elements
              const signUpBtn = document.getElementById('signUpBtn');
              const loginBtns = document.getElementById('loginBtns');
              const FormContainer = document.getElementById('FormContainer');

      Create Functions

      Now create two functions which help us to tell what happens when someone clicks on the buttons.

      First function is called signUp() which is triggered when someone clicks on the sign up button. 

                    
                function SignUp() {
                }

      Now inside of this function add a form structure which includes five input tags and one submit button.

      This structure exists inside of a container div called FormContainer.
      Now after that add some style which is fired when the sign up button is clicked. 

                      
                  function SignUp() {
                 FormContainer.innerHTML = `
                 <form method="get" accept-charset="utf-8">
                 <br> <br>
                 <input type="text" placeholder="Name" required="required" />
                 <input type="email" placeholder="Email" required="required" />
                 <input type="number" placeholder="Mobile" required="required"                               />
                 <input type="password" placeholder="Password" required="required" />
                 <input type="password" placeholder="Confirm Password" required="required" />
                 <br>
                 <button type="submit">Submit</button>
                 </form>
                 `;
                 FormContainer.style.background = '#00b3ef';
                 signUpBtn.style.background = '#00b3ef';
                 loginBtns.style.background = 'white';
                 }

      Second function is called Login() which is triggered when someone clicks on the Login button. 

                      
                  function Login() {
                  }

      Now inside of this function add a form structure which includes two input tags and one submit button.

      This structure exists inside of a container div called FormContainer.
      Now after that add some style which is fired when the Login button is clicked. 

                        
                    function Login() {
                 FormContainer.innerHTML = `
                 <form method="get" accept-charset="utf-8">
                 <br> <br>
                 <input type="email" placeholder="Email" required="required" />
                 <input type="password" placeholder="Password" required="required" />
                 <br>
                 <button type="submit">Submit</button>
                 </form>
                 `;
                 FormContainer.style.background = '#00b3ef';
                 signUpBtn.style.background = 'white';
                 loginBtns.style.background = '#00b3ef';
                 }

      Add click event on buttons

      After completing all the stuff the final step is to create a script which enables the button for action. Means add click events on the button tags. And called the two functions first SignUp() function and second Login() function.

                        
                    signUpBtn.addEventListener('click', SignUp);
                    loginBtns.addEventListener('click', Login);

      Now called the SignUp() function by default when the page is loaded.

      SignUp();
