Introducing MERN Stack

Learn MERN stack if you want to become a full stack web developer fast. The best parts of the MERN stack only use a single programming language called Javascript.

MERN stands for MongoDB(Database),Express js, React js, Node js.

Web Developer vs Web Designer

Web developer work is to build and maintain the websites and they have better knowledge about these languages HTML, CSS, Javascript.

And also they know about React js, Angular js, JQuery, sass, Bootstrap, gits and github.

Web developers used visual studio code, sublime Text editor..etc.

And on the other hand web designers have worked to design and create relational databases and create the web's layout for web developers.

Web Designers have these skills in graphic, design, color patterns, layout and logo design...etc.

They used these technologies for creating stunning designs like Adobe Photoshop, Illustration.
